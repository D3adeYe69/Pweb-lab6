# Pweb-lab6

Recipe Box — Front-end Lab 6

Quick start:

```powershell
npm install
npm run dev
```

Features planned:
- Create / edit / delete recipes (ingredients, markdown steps, tags, times, servings)
- Like/favorite recipes, filter/search by title/tag/ingredient
- Import from Markdown/CSV (later)
- Light/dark theme, local persistence (localStorage)

Branch plan: `feature/recipes-init` (scaffold + checkpoints)
